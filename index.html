Features

The full AI pipeline,
built for students

From PDF parsing to cosine similarity — every layer of the stack working for you.

📄
Smart Resume Parsing
PyPDF2 reads your PDF. NLTK cleans the text. Regex pattern matching extracts skills, tools, and experience automatically.
🤖
TF-IDF Matching
scikit-learn vectorizes your skill profile and all internship listings. Cosine similarity returns a precise fit score for each.
📊
Pandas + NumPy Analytics
Match scores injected into a pandas DataFrame — sorted, filtered, and returned as clean JSON via your Flask API.
🏢
Company Portal
Companies post internships with required skill sets and instantly see AI-ranked student candidates sorted by match score.
⚡
Skill Gap Analysis
Know exactly which skills separate you from a 95% match. Get targeted learning recommendations per internship.
🛡️
Admin Panel
Full platform oversight — manage users, companies, flag content, and view real-time analytics from one control center.
